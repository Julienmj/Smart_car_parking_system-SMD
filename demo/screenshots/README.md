# Screenshots Guide

To add screenshots to your README.md:

1. **Take Screenshots**
   - Open your browser and navigate to `http://localhost/carparking/parking-system/`
   - Take screenshots of:
     - Landing page (index.html)
     - Client dashboard (after logging in as client)
     - Admin dashboard (after logging in as admin)
     - Registration page
     - Payment interface

2. **Save Screenshots**
   - Save all screenshots in the `screenshots/` folder
   - Use these exact names:
     - `landing-page.png`
     - `client-dashboard.png`
     - `admin-dashboard.png`
     - `registration.png`
     - `payment.png`

3. **Recommended Screenshot Settings**
   - Resolution: 1920x1080 or higher
   - Format: PNG for better quality
   - Browser: Chrome or Firefox in full screen mode
   - Make sure to capture the full interface

4. **Default Login Credentials**
   - Admin: admin@parking.com / admin123
   - You can create a test client account via registration

Once screenshots are added, they will automatically appear in the README.md file.
